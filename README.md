# Forget Me Now

## Pre-requisites

- asdf - `brew install coreutils curl git asdf`
  - Golang - `asdf plugin add golang`
  - NodeJS - `asdf plugin add nodejs`
- Atlas - `curl -sSf https://atlasgo.sh | sh`
- Graphviz - `brew install graphviz`
- PlantUML - `brew install plantuml`
